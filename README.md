# Lista POO
## 📚 Sobre o Projeto

Este repositório contém uma lista de exercícios práticos de **Programação Orientada a Objetos (POO)** em Python. Os exercícios abordam os principais conceitos da POO como classes, objetos, herança, encapsulamento, protocol, polimorfismo e abstração.

## Estrutura do Projeto

```
lista 2/
├── armazenamento/
│   ├── controller/
│   ├── models/
│   ├── repo/
│   ├── ui/
│   ├── dados_locais.txt
│   └── main.py
├── funcionario/
├── midiaedu/
├── notificação/
└── protocol/
```

## Como executar

```bash
python armazenamento/main.py
python funcionario/main.py
python midiaedu/main.py
python notificação/main.py
python protocol/main.py
```
